# Fonts — Vizaje-Nica

## Описание

Данный модуль содержит новую типографическую систему для обновленного интерфейса
Vizaje-Nica.

Были выбраны два шрифта:

- Jost
- Playfair Display

---

# Причина замены

Основная цель редизайна — приблизить визуальный стиль сайта к современным
premium beauty брендам:

- Dior Beauty
- Chanel Beauty
- Aesop
- Rhode
- Glossier

Предыдущая типографика выглядела слишком технической и не создавала ощущения
премиального продукта.

Новая комбинация позволяет разделить интерфейсную и эмоциональную часть сайта.

---

# Используемые шрифты

## Jost

Используется для:

- основного текста;
- карточек товаров;
- кнопок;
- меню;
- форм;
- элементов интерфейса;
- каталога;
- checkout;
- корзины.

Причины выбора:

- высокая читаемость;
- современный внешний вид;
- хорошая работа на мобильных устройствах;
- поддержка кириллицы и латиницы;
- визуально ближе к современным luxury брендам.

Подключенные веса:

- 300
- 400
- 500
- 600

---

## Playfair Display

Используется для:

- заголовков;
- hero-блоков;
- важных маркетинговых секций;
- страниц брендов;
- крупных текстовых акцентов.

Причины выбора:

- премиальный характер;
- ассоциация с luxury и fashion-индустрией;
- визуальная близость к Dior, Chanel и другим beauty брендам.

Подключенные веса:

- 400
- 500
- 600
- 700

---

# Архитектура использования

Для сохранения единого визуального стиля рекомендуется использовать следующие
правила.

## Интерфейс

Использовать:

Jost

Примеры:

- меню;
- кнопки;
- карточки товаров;
- формы;
- чекбокс;
- фильтры;
- каталог;
- корзина;
- checkout.

---

## Заголовки

Использовать:

Playfair Display

Примеры:

- H1
- H2
- hero секции
- брендовые блоки
- маркетинговые заголовки

---

# Текущая реализация

В проекте установлены базовые правила:

- body использует Jost;
- input использует Jost;
- button использует Jost;
- textarea использует Jost;
- h1-h6 используют Playfair Display.

Это позволяет автоматически применять новую типографику ко всем новым
компонентам.

---

# Важно

При создании новых компонентов рекомендуется придерживаться следующих принципов:

- не использовать дополнительные шрифты;
- не подключать Google Fonts;
- использовать только локальные .woff2 файлы;
- сохранять единый стиль всей системы.

Типографика является одной из ключевых частей редизайна Vizaje-Nica и должна
оставаться консистентной во всех новых интерфейсах.
