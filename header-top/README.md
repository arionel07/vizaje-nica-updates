# Header Top Bar

## Описание

Данный компонент представляет собой информационную панель, расположенную над
основным header сайта.

Компонент отображается на всех страницах и является первым элементом интерфейса,
который видит пользователь после загрузки сайта.

---

# Причина внедрения

Основная задача блока — повысить доверие пользователя и сразу показать ключевые
преимущества магазина.

Пользователь получает важную информацию ещё до взаимодействия с каталогом или
товарами.

---

# Реализовано

Текущая версия отображает:

- Бесплатная доставка от 500 MDL
- 100% оригинальная продукция

Поддерживается:

- Desktop
- Mobile
- RU
- RO

# Что можно улучшить в будущем

## Автоматическая ротация сообщений

Вместо одного сообщения показывать несколько преимуществ:

- Бесплатная доставка от 500 MDL
- 100% оригинальная продукция
- Более 12 магазинов в Кишинёве
- Возможность протестировать продукцию в магазинах
- Более 99 мировых брендов

Смена сообщений может происходить автоматически каждые несколько секунд.
