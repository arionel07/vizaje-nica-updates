# App Loader

## Описание

Данный модуль реализует экран загрузки сайта Vizaje-Nica.

Loader отображается сразу после открытия страницы и скрывается только после
полной загрузки всех ресурсов сайта.

Компонент является частью новой дизайн-системы Vizaje-Nica и отвечает за первое
впечатление пользователя.

---

# Причина внедрения

Основная задача Loader — сделать загрузку сайта визуально приятной и избежать
резкого появления контента.

Без загрузочного экрана пользователь может увидеть:

- скачки интерфейса;
- постепенную подгрузку блоков;
- мерцание изображений;
- резкое появление контента.

Loader скрывает эти моменты и делает переход к сайту плавным.

---

# Реализовано

## Полноэкранный экран загрузки

Loader отображается поверх всего сайта до завершения загрузки страницы.

Используется:

- белый фон;
- логотип Vizaje-Nica;
- анимированная линия загрузки.

---

## Анимация логотипа

Логотип плавно изменяет прозрачность и положение.

Цель:

создать ощущение живого интерфейса без агрессивных эффектов.

---

## Индикатор загрузки

Используется минималистичная горизонтальная линия.

Особенности:

- постоянное движение;
- спокойная анимация;
- соответствует общей дизайн-концепции проекта.

---

## Плавное скрытие

После события:

window.load

Loader:

1. запускает анимацию исчезновения;
2. плавно снижает прозрачность;
3. становится недоступным для взаимодействия;
4. полностью удаляется из DOM.

# Используемые файлы

## loader.php

Содержит HTML-разметку загрузочного экрана.

---

## loader.css

Содержит:

- стили Loader;
- анимацию логотипа;
- анимацию линии загрузки;
- эффект плавного исчезновения.

---

## loader.js

Содержит логику:

- ожидание window.load;
- запуск скрытия Loader;
- удаление Loader после завершения анимации.

---

## design/

Содержит скриншоты и видео работы компонента.

Для данного модуля рекомендуется использовать видео (MP4), так как анимация
является важной частью пользовательского опыта.

# Что необходимо проверить при интеграции

## Старый Preloader

В проекте может присутствовать старый элемент:

```html
<div class="preloader"></div>
```

Перед интеграцией необходимо проверить:

- используется ли он где-либо;
- есть ли связанные стили;
- есть ли связанный JavaScript.

Если старый preloader больше не используется, его рекомендуется удалить.

Текущая реализация использует только:

```html
<div id="appLoader"></div>
```
