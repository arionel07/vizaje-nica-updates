# Store Map / Store Locator

## Описание

Store Map — это полностью переработанная страница магазинов Vizaje-Nica,
созданная для замены старого решения с обычной встроенной картой.

Основная цель страницы — помочь пользователю быстро найти ближайший магазин,
повысить доверие к бренду через демонстрацию физических точек продаж и сделать
поиск магазинов удобным как на компьютере, так и на мобильных устройствах.

---

# Что реализовано

## Новый Store Locator

Создана отдельная современная страница магазинов.

Пользователь может:

- просматривать все магазины списком;
- выбирать магазин на карте;
- строить маршрут;
- получать контактную информацию магазина;
- переключаться между списком и картой на мобильных устройствах.

---

## Интерактивная карта

Вместо старого решения реализована полноценная интерактивная карта.

Используется:

- MapLibre GL
- OpenFreeMap Positron Style

Преимущества:

- полностью бесплатно;
- не требует API ключей;
- не требует оплаты за просмотры карты;
- отсутствуют лимиты Mapbox;
- отсутствуют расходы Google Maps API.

---

## Кастомные маркеры

Созданы собственные маркеры Vizaje-Nica.

Особенности:

- показывают номер магазина;
- при выборе превращаются в фирменный маркер "V";
- имеют плавные анимации;
- синхронизированы со списком магазинов.

---

## Glass Panel

Создана информационная панель поверх карты.

Отображает:

- название магазина;
- адрес;
- телефон;
- кнопку построения маршрута.

Панель появляется плавно после выбора магазина.

---

## Синхронизация списка и карты

При выборе карточки:

- выделяется карточка;
- выделяется маркер;
- карта плавно перемещается к магазину;
- обновляется информационная панель.

При выборе маркера происходит аналогичное поведение.

---

## Mobile Experience

Для мобильных устройств реализован отдельный UX.

Добавлены переключатели:

- Списком
- На карте

Это позволяет:

- не перегружать экран;
- быстрее находить магазины;
- комфортно использовать карту на телефоне.

---

## Построение маршрута

Для выбранного магазина создаётся ссылка на маршрут.

Сейчас используется:

Google Maps Directions

Пользователь может открыть навигацию напрямую со смартфона.

---

Сейчас магазины выводятся из существующей структуры проекта.

Необходимо убедиться, что:

- все магазины управляются через админку;
- изменение адреса автоматически обновляет карту;
- изменение телефона автоматически обновляет карту;
- изменение координат автоматически обновляет карту.

# Используемые технологии

Frontend:

- HTML
- CSS
- JavaScript

Карта:

- MapLibre GL
- OpenFreeMap Positron

Маршруты:

- Google Maps Directions

---

# Подключение

## CSS

```html
<link
	rel="stylesheet"
	href="https://unpkg.com/maplibre-gl@5.6.2/dist/maplibre-gl.css"
/>
```

## JavaScript

```html
<script src="https://unpkg.com/maplibre-gl@5.6.2/dist/maplibre-gl.js"></script>
```

После подключения библиотеки необходимо подключить:

```html
stores-map.css stores-map.js
```

# Структура файлов

stores-map.php

- HTML структура страницы магазинов.
- Карточки магазинов.
- Информационная панель карты.
- Переключатель списка и карты.

stores-map.css

- Визуальная часть страницы.
- Адаптивность.
- Glassmorphism панель.
- Кастомные маркеры.
- Анимации.

stores-map.js

- Инициализация карты.
- Работа маркеров.
- Синхронизация карты и списка.
- Работа мобильных вкладок.
- Построение маршрутов.

language.json

- Все тексты страницы.
- Поддержка RU и RO языков.

design/

- Скриншоты.
- Референсы.
- Демонстрация финального результата.
